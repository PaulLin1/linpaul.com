The website is built using the Warp framework in Rust, with random images

displayed using vanilla JavaScript. To enhance performance, images are both

preloaded and lazy loaded. For the CI/CD pipeline, I use GitHub Actions to build a

Docker container and deploy it through Fly.io. I chose Fly.io because it provides a

simple, cost-effective solution that perfectly fits the scale of this project.

The website’s design draws inspiration from art.yale.edu. In a UX class, a

classmate once dismissed it as an example of poor design, but I saw it differently.

I like how its eccentric, collage-like layout stands out over the duplicated image.

The chair’s unusual patterns allow colors to inhabit unexpected shapes, giving the

piece a lively, DIY feel. The bold, puke-like yellows were chosen with the same

deliberate care as the subtle eggshell whites found on a sleek tech site. It taught

me that sometimes the effort behind an idea matters more than the idea itself.
