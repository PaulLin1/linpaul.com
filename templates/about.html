The website is built using the Warp framework in Rust, with random images displayed using vanilla JavaScript. To improve performance, the images are preloaded and lazy loaded. For the CI/CD pipeline, I use GitHub Actions to build a Docker container and deploy it via Fly.io. I chose Fly.io because it offers a simple and cost-effective solution that fits the scale of this project well.
